# 二、有关保研的介绍
**保研**：被保送者不经过笔试等初试一些程序，通过一个考评形式鉴定学生的学习成绩、综合素质等，在一个允许的范围内，直接由学校保送读研究生。

21级开始保研评价体系分为三部分：90%成绩+5%综测+5%竞赛论文。

参考链接：[关于保研的介绍](https://zhuanlan.zhihu.com/p/259145735)

## 3.1 前期材料准备（大三上学期结束-明年6月份初）
**时间**：大三上学期结束到明年6月份

### 3.1.1 **必做工作：材料整理——本科阶段成果总结**：
1. **荣誉成果**
- 荣誉称号、奖学金：时间+奖项名称
- 竞赛：时间+奖项名称+等级+组委会名称
2. **论文**
- 格式：作者序号+论文名称+期刊名称+时间
- 论文简介+自己主要工作
3. **科研项目**
- 格式：时间+课题名称+课题内容+课题成果+你的收获
- 课题内容根据实际需求进行删减
4. **社会实践、学生工作**（不是太重要）
- 社会实践格式：时间+实践内容+实践成果
- 学生工作格式：时间+任职工作+自己贡献

### 3.1.2 **系统提交材料准备（夏令营、预推免提交材料类似）**：
1. **成绩单、成绩排名证明、四六级成绩证明**
  - 校务行小程序：4元1份
  - 学校的自助打印机
2. **获奖证书纸质版、扫描版**
  - 扫描工具推荐：全能扫描王
  - 大小一致：A4纸
  - 适当调整方向：适合观看
  - PDF、jpg格式均要有
3. **学生证、身份证、学籍在线报告**
  - 学生证扫描版：两页放一张A4纸
  - 身份证扫描版：正反面放一页A4纸
  - 学籍在线报告：学信网申请下载
4. **专家推荐信**
  - 副教授及以上职称
  - 以老师的口吻概括自己，建议从科研及课程导师着手
5. **个人陈述/个人自我介绍/研究生规划**
  - 建议写多个版本，1000字、500字、300字等
  - 从学业情况、竞赛科研、社会实践、未来规划四方面入手写
    - **第一部分：致谢**
      - 首先感谢您在百忙之中阅读我的个人陈述。我叫***，现就读于***，是***的学生。怀着对贵校严谨的治学态度和浓厚学术氛围的崇敬与渴望，申请贵校举办的推免面试，下面我将从教育背景、科研竞赛、课外实践、未来规划等四个方面简单介绍一下自己。
    - **第二部分：教育背景**
      - 建议基本情况+能力。例如：前六学期平均绩点4.38，大三学年专业排名：1/97。单学期最高绩点4.78，刷新我校单学期最高纪录。在所修的课程中，68门课程90分以上，44门课程接近满分。在学习中，我注重知识的理解与延伸，热衷于将不同学科内容结合形成知识体系，具备较为扎实的专业基础，有能力完成研究生阶段理论课程的学习。
    - **第三部分：科研竞赛**
      - 建议基本情况+能力。自己目前有哪些成果，它跟你以后的研究生生活有什么关联，或者能为你提供什么。
    - **第四部分：课外实践**（根据自己的材料适当删减）
      - 这一部分可有可无，根据个人情况进行删减。
    - **第五部分：未来规划**
      - 谈一下自己本科阶段感兴趣的领域，以及未来打算攻读的研究方向。下一步的学习计划，未来的研究志向。
    - 格式按照书信的格式撰写
6. **个人简历**
  - 控制在一页A4以内，切忌两页
  - 简洁、美观
  - 从基本情况、教育背景、科研竞赛、近期工作经理等几个方面着手
  - 根据自己的材料适当详写或略写

    - 若竞赛多，选取与专业关联性比较大的写

    - 项目名称，负责工作，开展时间
    - 基于什么方法、做了什么东西，有什么成果

8. **其他证明材料**（科研项目证明、发表的论文、社会实践证明材料等）

## 3.2 夏令营（6月份-7月份末）
- 时间：6月份-7月份末
- 针对人群：rk前5%机会较大，其他同学要多去尝试
- 难度：相较于预推免，难度稍微大一些，冲top2较难

## 3.3 预推免（8月份末-9月份末）
- 时间：8月份末-9月份末
- 针对人群：具有保研资格的同学
- 难度：有机会冲top2/华五

## 3.4 考核形式及内容
### 3.4.1 面试考核
1. **英文考核**
  - 英文自我介绍1分钟、3分钟
  - 英文问答，偏日常，专业课也有涉及
  - 英文翻译，抽题形式，论文/日常
  - 听力测试，微信问卷的形式，四六级题型
2. **专业课考核**
  - 数理考核，高等数学、线性代数、概率论
    - 偏基础的内容，什么是矩阵的特征值等
  - 专业课考核，计算机：408
    - TCP/UDP协议的优缺点
    - 死锁机制等
3. **综合考核：项目、竞赛、论文考核**
  - 根据考生所述的经历进行随机提问
  - 例如，机器学习、深度学习项目常问
    - 预处理做了什么
    - 数据集有多大
    - 什么是过拟合，怎么解决过拟合问题
    - 什么是SVM，关于SVM的知识点
    - 池化层的原理是什么
  - 数学建模
    - 你在建模过程中负责什么工作
    - 为什么用这个方法，与其他方法相比你们的优点在哪里
  - 计算机设计大赛
    - 这个是怎么实现的
    - 你们为什么从这个角度去考虑
    - 你负责什么内容
4. **思政考核**
  - 作为当代大学生，对××的理解
  - 对××的看法
5. **导师组考核**
  - 看论文汇报。课题组发给你几篇文献，并规定好时间，针对内容提问
  - 导师组面试。针对本科经历，主要科研项目及论文
